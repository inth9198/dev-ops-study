# macOS에서 시스템 콜 추적하기 (dtruss 사용법)

## 올바른 명령어:

# 1. 화면에 출력
sudo dtruss ./hello

# 2. 파일로 저장
sudo dtruss ./hello > hello.log 2>&1

# 3. 특정 시스템 콜만 필터링
sudo dtruss -t open,read,write ./hello

# 4. 실행 중인 프로세스 추적
sudo dtruss -p <PID>

## 주의사항:
- dtruss는 관리자 권한(sudo)이 필요합니다
- System Integrity Protection (SIP)이 켜져 있으면 일부 기능이 제한될 수 있습니다
- -o 옵션은 출력 파일이 아닙니다 (잘못된 사용법)
